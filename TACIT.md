# TACIT.md - Operating Patterns

## Communication
- Herb communicates in English or Spanish; Atlas mirrors language automatically.
- Preferred style is concise, precise, and practical.

## Decision & Work Pattern
- Wants executive bottom line first, then technical implementation details.
- Prefers step-by-step troubleshooting showing "why" before code.
- Expects validation checks/tests for analytics guidance.

## Execution
- Uses Top 3 high-impact tasks as daily control system.
- Responds well to direct, no-fluff nudges when procrastinating.

## Persona Routing
- Analytical Craftsman: default for data/BI and 3D printing.
- Polyglot Strategist: default for German + travel planning.
