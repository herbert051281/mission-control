## Qué se mejoró (V2)
- Flujo narrativo más ejecutivo: **resumen → ruta → agenda → costos clave → opciones de decisión → riesgos → cierre**.
- Menos texto denso y mayor legibilidad visual (bloques limpios, tablas compactas, mensajes directos).
- Se mantiene tema visual Perú (paleta rojo/tierra + fotografías de Perú).
- Todas las cifras monetarias están señaladas explícitamente como **estimadas / referenciales**.

## Nuevas diapositivas incorporadas
- **Opciones recomendadas**
  - 3 escenarios: **Ahorro, Balanceado, Confort**
  - Incluye rango total estimado por escenario:
    - Ahorro: **USD 3,750–4,600**
    - Balanceado: **USD 4,900–5,900**
    - Confort: **USD 6,200–7,050**
- **Riesgos y mitigación**
  - Riesgos cubiertos: **altitud, traslados, fatiga infantil, clima/logística**
  - Enfoque tabla ejecutiva: probabilidad, impacto y mitigación accionable.
