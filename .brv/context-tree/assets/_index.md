# Domain: assets

## Overview
The `assets` domain manages media resources, licenses, and source attributions, primarily focusing on travel-related photography and metadata.

### Peru Travel Resource Management (`peru_travel/`)
*   **Storage & Metadata**: Photographic assets are stored in `assets/peru/` with centralized mapping and license data in `assets/peru/sources.json`.
*   **Compliance**: Mandatory mapping of every image to a source URL and license in `sources.json`.
*   **Licensing**: Tracks Wikimedia Commons origins under CC BY-SA 4.0, CC BY 3.0, and CC0 protocols.

### Key Components
*   **image_assets_and_sources**: Technical documentation of specific image licenses and origins.
*   **context**: Tracking index for photographic assets used in travel documentation.
